Введение

Это пример семантической вёрстки с использованием разнообразных HTML-элементов. Здесь есть акценты, выделения, инлайн-код и даже внешняя ссылка.

HTML — это стандартный язык разметки для создания веб-страниц. Он описывает структуру контента, а не его оформление.

Выделение текста

В тексте можно использовать жирный для важного, курсив для акцента, подчёркнутый для особого внимания, зачёркнутый для удалённого, вставленный для нового, маленький текст, верхний и нижний индексы.
